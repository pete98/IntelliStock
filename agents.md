- **Entry Points**: `index.js` registers the `App` component (`App.js`) as root via Expo.
- **UI Shell**: `App.js` renders a placeholder view with centered text and `StatusBar`.
- **Project Config**: `app.json` defines Expo metadata (name, icons, splash, orientation) with new architecture enabled.
